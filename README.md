# 🛍️ Superstore Sales Dashboard

## 📌 Project Overview

This project presents an interactive sales dashboard built to analyze the performance of a retail superstore.
It provides insights into sales, profit, customer segments, shipping modes, and regional performance.

## 🎯 Objectives

* Analyze overall business performance

* Identify high and low-performing products

* Evaluate regional profitability

* Understand customer purchasing behavior

## 📊 Key Metrics

* 💰 Total Sales: $2.30M
* 📈 Total Profit: $286.40K
* 📉 Profit Margin: 12.47%
* 📦 Total Quantity Sold: 37.87K

## 📈 Insights

* 🌍 West region generates the highest profit and margin
* ⚠️ Central region has the lowest profitability

* 🪑 Tables category is causing losses

* 📱 High-profit products include Copiers and Phones

* 🚚 Standard shipping is the most frequently used mode

## 🛠 Tools Used

* Power BI
* Data Visualization
